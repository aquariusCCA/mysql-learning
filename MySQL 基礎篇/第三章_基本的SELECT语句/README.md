# 第三章_基本的SELECT语句

這一章開始正式進入 SQL 查詢語句。目前已整理完成前 3 節，先建立 SQL 的整體框架，再補上書寫規則與基本 `SELECT` 查詢語法。

## 建議閱讀順序

1. 先讀 SQL 概述，理解 SQL 在關係型資料庫中的角色。
2. 再讀 SQL 語言的規則與規範，熟悉大小寫、註解、命名與匯入指令。
3. 再讀基本的 `SELECT` 語句，掌握查常量、查全部列、查指定列、列別名、去重與 `NULL` 運算。
4. 把 `DDL`、`DML`、`DCL`、`DQL`、`TCL` 的分類和實際查詢寫法一起建立起來。

## 本章適合回查的主題

- 想快速回想 SQL 是什麼、歷史背景與常見分類時，優先看第 1 節。
- 想確認 SQL 的大小寫、註解、命名或 `source` 匯入方式時，優先看第 2 節。
- 想回查 `SELECT *`、列別名、`DISTINCT`、`NULL` 運算、着重號或查詢常數時，優先看第 3 節。
- 之後若補齊更多 `SELECT` 查詢內容，這一章會變成查詢語法的主導航入口。
