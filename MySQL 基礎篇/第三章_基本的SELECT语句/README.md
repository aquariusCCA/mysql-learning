# 第三章_基本的SELECT语句

這一章開始正式進入 SQL 查詢語句。目前已整理完成前 2 節，先建立 SQL 的整體框架，再補上書寫規則與基本規範。

## 建議閱讀順序

1. 先讀 SQL 概述，理解 SQL 在關係型資料庫中的角色。
2. 再讀 SQL 語言的規則與規範，熟悉大小寫、註解、命名與匯入指令。
3. 把 `DDL`、`DML`、`DCL`、`DQL`、`TCL` 的分類和實際書寫習慣一起建立起來。
4. 後續新增 `SELECT` 相關小節時，再依照章節順序往下接。
