# 第三章_基本的SELECT语句

这一章开始正式进入 SQL 查询语句。内容会先建立 SQL 的整体框架，再补上书写规则、基本 `SELECT` 查询语法、查看表结构的方式，以及 `WHERE` 条件过滤的基础用法。

## 本章在整体学习地图中的位置

- 这一章是从“认识 SQL”过渡到“开始真正写查询语句”的起点。
- 如果前两章主要是在建立数据库概念和准备 MySQL 环境，这一章就开始进入最常用、最需要反复练习的查询语法。
- 读完本章后，建议先进入 [第四章_运算符](../第四章_运算符/README.md)，把条件判断与表达式细节补齐，再继续串回查询练习。

## 建议阅读顺序

1. [1 SQL 概述](./1%20SQL%20概述.md)：先理解 SQL 是什么，以及 `DDL`、`DML`、`DCL`、`DQL`、`TCL` 的基本分类。
2. [2 SQL语言的规则与规范](./2%20SQL语言的规则与规范.md)：再熟悉大小写、注释、命名与 `source` 导入等基础规范。
3. [3 基本的 SELECT 语句](./3%20基本的%20SELECT%20语句.md)：开始掌握 `SELECT`、列别名、`DISTINCT`、`NULL` 运算和查询常数。
4. [4 显示表结构](./4%20显示表结构.md)：学会在查询前先确认字段、类型、默认值与索引信息。
5. [5 过滤数据](./5%20过滤数据.md)：掌握 `WHERE` 子句如何限制查询结果。

## 本章小节

- [1 SQL 概述](./1%20SQL%20概述.md)：建立 SQL 的背景、标准与分类框架。
- [2 SQL语言的规则与规范](./2%20SQL语言的规则与规范.md)：整理 SQL 的结束符、大小写、注释、命名和导入规范。
- [3 基本的 SELECT 语句](./3%20基本的%20SELECT%20语句.md)：进入最基础的查询写法与常见细节。
- [4 显示表结构](./4%20显示表结构.md)：学习如何快速查看表结构和字段信息。
- [5 过滤数据](./5%20过滤数据.md)：学习如何通过条件筛选需要的数据行。

## 本章适合回查的主题

- 想快速回想 SQL 是什么、常见分类怎么分：看 [1 SQL 概述](./1%20SQL%20概述.md)
- 想确认 SQL 的大小写、注释、命名或 `source` 导入方式：看 [2 SQL语言的规则与规范](./2%20SQL语言的规则与规范.md)
- 想回查 `SELECT *`、列别名、`DISTINCT`、`NULL` 运算或查询常数：看 [3 基本的 SELECT 语句](./3%20基本的%20SELECT%20语句.md)
- 想确认表中有哪些字段、字段类型、是否允许 `NULL`、是否有主键或索引：看 [4 显示表结构](./4%20显示表结构.md)
- 想回查 `WHERE` 子句位置，或想按条件筛选查询结果：看 [5 过滤数据](./5%20过滤数据.md)
