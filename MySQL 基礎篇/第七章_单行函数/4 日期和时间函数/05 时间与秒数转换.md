# 4.5 时间与秒数转换

> 所属章节：[第七章_单行函数 / 4 日期和时间函数](./README.md)
> 关键字：TIME_TO_SEC、SEC_TO_TIME
> 建议回查情境：需要把时间换算成总秒数，或把秒数还原成 `HH:MM:SS` 时

## 本节导读

这一节只有两個函数，但它们非常实用：一个把时间转换成秒数，适合做比较和计算；另一个把秒数还原成可读时间，适合显示结果。

## 函数表

| 函数 | 用法 |
| --- | --- |
| `TIME_TO_SEC(time)` | 将 `time` 转化为秒并返回，公式为 `小时 * 3600 + 分钟 * 60 + 秒` |
| `SEC_TO_TIME(seconds)` | 将 `seconds` 转化为包含小时、分钟和秒的时间 |

## 示例

```sql
SELECT TIME_TO_SEC(NOW());
+--------------------+
| TIME_TO_SEC(NOW()) |
+--------------------+
|              78774 |
+--------------------+
1 row in set (0.00 sec)
```

```sql
SELECT SEC_TO_TIME(78774);
+--------------------+
| SEC_TO_TIME(78774) |
+--------------------+
| 21:52:54           |
+--------------------+
1 row in set (0.12 sec)
```

## 使用提醒

- 做时间长度比较时，先转成秒数通常更方便。
- 展示时再转回 `HH:MM:SS`，可读性会更高。
