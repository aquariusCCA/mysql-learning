# 4 日期和时间函数

这一组笔记整理 MySQL 中最常见的日期与时间函数。内容不只包括“拿当前时间”和“格式化日期”，也包含时间戳转换、日期组成部分提取、时间加减、秒数换算，以及日期解析等实用场景。

## 这组笔记适合怎么读

- 第一次学习时，建议先读 [01 获取当前日期与时间](./01%20获取当前日期与时间.md)，建立最基础的当前时间函数认识。
- 接着读 [02 日期与时间戳的转换](./02%20日期与时间戳的转换.md) 和 [03 获取月份星期天数等信息](./03%20获取月份星期天数等信息.md)，先掌握查询和拆解日期值的常见方法。
- 之后再读 [04 日期字段提取与 EXTRACT](./04%20日期字段提取与%20EXTRACT.md)、[05 时间与秒数转换](./05%20时间与秒数转换.md) 和 [06 日期时间的加减与计算](./06%20日期时间的加减与计算.md)。
- 最后读 [07 日期格式化与解析](./07%20日期格式化与解析.md)，把“日期怎么显示”和“字符串怎么转回日期”这一层补齐。

## 本组小节

- [01 获取当前日期与时间](./01%20获取当前日期与时间.md)：整理 `CURDATE()`、`CURTIME()`、`NOW()`、`SYSDATE()`、`UTC_DATE()`、`UTC_TIME()`。
- [02 日期与时间戳的转换](./02%20日期与时间戳的转换.md)：整理 `UNIX_TIMESTAMP()` 与 `FROM_UNIXTIME()`。
- [03 获取月份星期天数等信息](./03%20获取月份星期天数等信息.md)：整理 `YEAR()`、`MONTH()`、`DAY()`、`WEEK()`、`DAYOFYEAR()` 等拆解函数。
- [04 日期字段提取与 EXTRACT](./04%20日期字段提取与%20EXTRACT.md)：集中说明 `EXTRACT(type FROM date)` 的用法与可提取部分。
- [05 时间与秒数转换](./05%20时间与秒数转换.md)：整理 `TIME_TO_SEC()` 和 `SEC_TO_TIME()`。
- [06 日期时间的加减与计算](./06%20日期时间的加减与计算.md)：整理 `DATE_ADD()`、`DATE_SUB()`、`DATEDIFF()`、`TIMEDIFF()`、`LAST_DAY()`、`MAKEDATE()` 等函数。
- [07 日期格式化与解析](./07%20日期格式化与解析.md)：整理 `DATE_FORMAT()`、`TIME_FORMAT()`、`GET_FORMAT()`、`STR_TO_DATE()` 和常用格式符。

## 适合快速回查的主题

- 想知道“今天几号”“现在几点”“当前系统时间是什么”：看 [01 获取当前日期与时间](./01%20获取当前日期与时间.md)
- 想把普通时间和 Unix 时间戳互转：看 [02 日期与时间戳的转换](./02%20日期与时间戳的转换.md)
- 想取出年份、月份、星期、季度或一年中的第几天：看 [03 获取月份星期天数等信息](./03%20获取月份星期天数等信息.md)
- 想直接提取复合日期字段，例如 `MINUTE_SECOND`：看 [04 日期字段提取与 EXTRACT](./04%20日期字段提取与%20EXTRACT.md)
- 想做日期加减、时间加减、日期差值或最近 7 天统计：看 [06 日期时间的加减与计算](./06%20日期时间的加减与计算.md)
- 想把日期格式化成指定字符串，或把字符串解析成日期：看 [07 日期格式化与解析](./07%20日期格式化与解析.md)
