# 第七章_单行函数练习题

这份题单只基于第七章现有笔记内容设计，重点练习：

- 单行函数与聚合函数的区别
- 数值、字符串、日期时间函数的基本使用
- `IF()`、`IFNULL()`、`CASE()` 的条件处理
- 加密与解密函数、信息函数与其他工具函数
- 在真实查询中把函数放进 `SELECT` 与 `WHERE`

## 使用说明

- 练习数据默认使用仓库中的 `資料/atguigudb.sql`。
- 除题目特别说明外，优先基于 `employees` 表完成练习。
- 这一份只放题目，不放答案；建议先独立完成，再回到第七章各小节对照。
- 涉及“判断差异”或“改错”的题目，建议先写出原因，再写最终 SQL。

## 基础题

### 1. 函数分类题

说明什么是单行函数，并对比它和聚合函数在下面两个角度上的差异：

- 输入数据范围
- 输出结果个数

### 2. 分类判断题

请判断下面这些函数中，哪些属于单行函数，哪些不属于：

- `UPPER()`
- `ROUND()`
- `NOW()`
- `COUNT()`
- `AVG()`
- `IFNULL()`

### 3. 数值函数识别

请分别说明下面这些数值函数各适合处理什么问题：

- `ABS()`
- `CEIL()`
- `FLOOR()`
- `ROUND()`
- `TRUNCATE()`
- `RAND()`

### 4. `ROUND()` 与 `TRUNCATE()` 比较

说明 `ROUND(x, y)` 和 `TRUNCATE(x, y)` 的差别，并写出下面表达式各自的结果：

- `ROUND(12.343, 2)`
- `TRUNCATE(12.343, 2)`
- `ROUND(12.324, -1)`
- `TRUNCATE(12.324, -1)`

### 5. 随机数题

说明 `RAND()` 和 `RAND(x)` 的差别，并解释为什么相同种子的 `RAND(x)` 结果可以重复。

### 6. 字符串长度题

说明 `CHAR_LENGTH()` 和 `LENGTH()` 的差别，并回答为什么在中文场景下这两个函数的结果可能不同。

### 7. 拼接题

请分别写出：

1. 使用 `CONCAT()` 拼接员工 `last_name` 和 `job_id`；
2. 使用 `CONCAT_WS('-', ...)` 拼接同样内容。

并说明两者差别。

### 8. 大小写转换题

查询员工的 `last_name`，同时额外显示：

- 全大写版本
- 全小写版本

### 9. 截取与定位题

请分别说明下面几个函数的用途，并各举一个简单示例：

- `LEFT()`
- `RIGHT()`
- `SUBSTR()`
- `LOCATE()`

### 10. 去空格与补齐题

说明 `TRIM()`、`LTRIM()`、`RTRIM()`、`LPAD()`、`RPAD()` 分别适合什么场景。

## 进阶题

### 11. 字符串替换题

- 一个是按什么维度替换；
- 另一个是按什么维度替换。

### 12. `FIELD()` 与 `FIND_IN_SET()` 比较

- `FIELD(s, s1, s2, ...)`
- `FIND_IN_SET(s1, s2)`

并解释为什么它们虽然都能“找位置”，但查找来源不同。

### 20. `IF()` 与 `IFNULL()` 比较

### 28. 字符串函数实作题

- 姓名长度
- 姓名前 `3` 个字符

要求至少使用 `CHAR_LENGTH()` 或 `LENGTH()`、`LEFT()` 或 `SUBSTR()`、`LOCATE()` 中的两个函数。

### 29. 条件标签题

使用 `CASE WHEN`，按下面规则给员工工资打标签：

- `salary >= 15000` 显示 `高薪`
- `salary >= 10000` 显示 `潜力股`
- `salary >= 8000` 显示 `一般`
- 其他显示 `基础`

### 30. 状态映射题

要求：

- 你自己设定 `1` 到 `4` 的状态含义；
- 至少写出一个 `ELSE` 分支。

### 31. 年薪计算题

查询员工的：

- `employee_id`
- `salary`
- `commission_pct`
- 年薪字段 `annual_salary`

要求：

- 年薪按 `12 * salary * (1 + IFNULL(commission_pct, 0))` 思路计算；
- 必须体现 `IFNULL()` 的用法。

### 32. 日期处理综合题

### 36. 综合题：从需求到 SQL

完成下面需求：

- 一个格式化后的姓名字段：转成大写；

要求：
