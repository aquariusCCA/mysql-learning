# 第五章_排序与分页

这一章开始整理 MySQL 查询结果的顺序控制与分页处理。排序用于让查询结果按指定字段稳定展示；分页则用于控制一次返回多少数据，以及从哪一段结果开始返回。

## 本章在整体学习地图中的位置

- 第三章主要建立了 `SELECT`、`WHERE` 和表结构查看的基础查询骨架。
- 第四章补上了表达式和条件判断所需的运算符。
- 这一章开始处理查询结果“怎么展示”的问题，也就是排序和分页，把“查什么”进一步推进到“怎么稳定展示、怎么分段返回”。

## 建议阅读顺序

1. [1 排序数据](./1%20排序数据.md)：先掌握 `ORDER BY` 的基本写法、升降序、单列排序和多列排序。
2. [2 分页](./2%20分页.md)：接着理解 `LIMIT`、偏移量、页码公式，以及分页与排序的关系。

## 本章小节

- [1 排序数据](./1%20排序数据.md)：整理 `ORDER BY` 的基本规则、别名排序、位置排序和多列排序。
