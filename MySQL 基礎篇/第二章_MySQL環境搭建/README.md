# 第二章_MySQL環境搭建

这一章把“概念认识”推进到“环境可用”。内容覆盖 MySQL 的下载、安装、配置、登录、基础演示、图形化工具、目录结构与常见问题处理，目标是把后续 SQL 学习需要的操作环境先搭起来。

## 本章在整体学习地图中的位置

- 第一章解决的是数据库概念和基础名词，这一章开始进入真正可操作的 MySQL 环境。
- 如果这章没有读顺，后面学 `SELECT`、运算符或排序时，容易卡在安装、服务启动、字符集或连接工具问题上。
- 读完本章后，通常会接着进入 [第三章_基本的SELECT语句](../第三章_基本的SELECT语句/README.md)，开始正式写 SQL 查询。

## 本章小节

- [1 MySQL 的卸载](./1%20MySQL%20的卸载.md)：处理重装前清理服务、残余文件、注册表与环境变量的问题。
- [2 MySQL 的下载、安装、配置](./2%20MySQL%20的下载、安装、配置.md)：涵盖版本选择、安装向导、服务配置、环境变量与安装失败排查。
- [3 MySQL 的登录](./3%20MySQL%20的登录.md)：整理服务启停、命令行登录、版本查看与退出方式。
- [4 MySQL 演示使用](./4%20MySQL%20演示使用.md)：通过创建数据库、建表、插入数据与查看创建信息，建立第一次实际操作感。
