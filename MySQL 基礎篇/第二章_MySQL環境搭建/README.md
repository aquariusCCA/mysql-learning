# 第二章_MySQL環境搭建

這一章主要整理 MySQL 在 Windows 環境下的安裝、設定、登入、基本使用與常見問題處理，內容偏操作型，適合邊看邊實作。

## 建議閱讀順序

1. 先看下載、安裝與初始化設定，建立完整的安裝流程。
2. 再學會如何登入 MySQL，並確認服務是否正常運作。
3. 接著用演示操作熟悉建立資料庫、建立資料表與查詢資料的基本流程。
4. 最後再補圖形化工具、目錄結構與常見問題，當成日後回查的工具頁。
