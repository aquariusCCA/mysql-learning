# 第二章_MySQL環境搭建

這一章主要整理 MySQL 在 Windows 環境下的卸載、下載安裝、初始化配置、登入、基本使用、圖形化工具、目錄結構與常見問題處理。

內容偏操作型，第一次學習時建議邊看邊實作；之後回查時，可以直接按「安裝 / 登入 / 使用 / 排錯」定位到對應小節。

## 本章主要內容

- MySQL 卸載與重裝前的清理流程。
- MySQL8.0 的下載、安裝、初始化配置與環境變量設定。
- MySQL 服務啟停、命令行登入與退出方式。
- 基礎 SQL 操作演示，包括建立資料庫、資料表、插入與查詢資料。
- Workbench、Navicat、SQLyog、DBeaver 等圖形化管理工具。
- MySQL 安裝目錄、資料目錄、配置檔與源碼目錄的基本認識。
- 初學階段常見錯誤與排查方式。
