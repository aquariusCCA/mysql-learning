这一节主要介绍在 MySQL 安装完成后，如何确认服务已经启动，以及如何通过图形界面或命令行登录并退出 MySQL。

这篇内容很适合当作命令行回查页。第一次学习时，建议按“先确认服务状态 -> 再执行登录 -> 最后记住退出方式”的顺序阅读；如果你只是忘记登录命令格式，可以直接跳到 `3.2 自带客户端的登录与退出`。

## 关键字

- `MySQL 服务`：登录前需先确认服务已启动
- `services.msc`：Windows 服务管理入口
- `MySQL80`：Windows 下常见的 MySQL 服务名
- `net start`：启动 MySQL 服务
- `net stop`：停止 MySQL 服务
- `mysql -h -P -u -p`：命令行登录 MySQL 的基本格式
- `localhost` `127.0.0.1`：本机连接常用主机地址
- `3306`：MySQL 默认端口
- `root`：默认常见管理员账户
- `mysql -V` `mysql --version`：查看客户端版本
- `select version()`：登录后查看服务端版本
- `exit` `quit`：退出 MySQL 登录

## 3.1 服务的启动与停止

MySQL 安装完成之后，需要先启动服务器进程，否则客户端无法连接数据库。

在前面的安装配置过程中，MySQL 已经被安装为 Windows 服务。如果你当时勾选了开机自启动，那么 Windows 启动时也会自动启动 MySQL。

### 方式 1：使用图形界面工具

先打开 Windows 服务管理界面。常见入口有 4 种：

1. 在“计算机”上右键，进入【管理】->【服务和应用程序】->【服务】
2. 打开【控制面板】->【系统和安全】->【管理工具】->【服务】
3. 在任务栏右键，打开【任务管理器】->【服务】
4. 在开始菜单搜索框输入 `services.msc`，按 Enter

打开服务列表后，找到 `MySQL80`，右键即可选择启动或停止。

![在 Windows 服务中启动或停止 MySQL](./images/image-20211014183908375.png)

### 方式 2：使用命令行工具

如果你更习惯在命令行里处理服务，可以使用下面的命令：

```bash
# 启动 MySQL 服务
net start MySQL服务名

# 停止 MySQL 服务
net stop MySQL服务名
```

![使用命令行启动或停止服务](./images/image-20211014184037414.png)

### 使用命令行启停服务时的注意点

1. `start` 和 `stop` 后面的服务名，必须和你安装时配置的服务名一致，课程里的常见名称是 `MySQL80`。
2. 如果执行命令后提示“拒绝服务”，通常需要使用管理员身份重新打开命令提示符，再执行一次。

## 3.2 自带客户端的登录与退出

当 MySQL 服务已经启动后，就可以使用客户端连接 MySQL。

### 登录方式 1：MySQL 自带客户端

可以通过开始菜单进入：

```text
开始菜单 -> 所有程序 -> MySQL -> MySQL 8.0 Command Line Client
```

![打开 MySQL 自带客户端](./images/image-20211014184425147.png)

> ✏️ 这种方式通常直接以 `root` 用户登录，更适合作为初学阶段的快速入口。

### 登录方式 2：Windows 命令行

你也可以直接在 Windows 命令行中执行登录命令。

![在命令行中登录 MySQL](./images/image-20211014185035137.png)

基本格式如下：

```bash
mysql -h 主机名 -P 端口号 -u 用户名 -p密码
```

示例：

```bash
mysql -h localhost -P 3306 -u root -pabc123
```

上面这个例子表示：

- `-h localhost`：连接本机
- `-P 3306`：连接 MySQL 默认端口
- `-u root`：使用 `root` 用户登录
- `-pabc123`：使用 `abc123` 作为密码

### 命令行登录时的注意点

1. `-p` 和密码之间不能有空格；其他参数名和参数值之间可以有空格，也可以不写空格。

   ```bash
   mysql -hlocalhost -P3306 -uroot -pabc123
   ```

2. 更推荐的写法是只输入 `-p`，然后在下一行手动输入密码，这样更安全。

   ```bash
   mysql -h localhost -P 3306 -u root -p
   Enter password: ****
   ```

3. 如果客户端和服务器在同一台机器上，那么主机名可以写 `localhost` 或 `127.0.0.1`。

4. 如果你连接的是本机，而且端口号没有改过，那么 `-h localhost` 和 `-P 3306` 通常都可以省略，常见简写如下：

   ```bash
   mysql -u root -p
   Enter password: ****
   ```

5. 连接成功后，命令行会显示 MySQL Server 的版本信息，以及这次连接对应的会话 `id`。

### 如何查看 MySQL 版本

如果你还没有登录，可以在命令行直接查看客户端版本：

```bash
mysql -V
mysql --version
```

如果你已经登录，也可以在 MySQL 命令行中执行：

```sql
select version();
```

## 3.3 退出登录

退出 MySQL 登录可以使用下面任一命令：

```bash
exit
# 或
quit
```
