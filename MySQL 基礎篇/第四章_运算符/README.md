# 第四章_运算符

这一章开始整理 MySQL 中的运算符。运算符是后续编写查询条件、计算表达式、筛选数据与组合逻辑判断的基础。

目前已整理第 1 至 2 节，先从算术运算符开始理解数值计算，再进入比较运算符，掌握 `WHERE` 条件中常用的相等、空值、范围、列表与字符串匹配判断。

## 本章主要内容

- 算术运算符的基本写法。
- 加法、减法、乘法、除法与求模的结果特点。
- MySQL 中 `+` 与字符串拼接的区别。
- 除法、`DIV`、除以 `0` 与 `NULL` 的关系。
- 使用 `MOD` 处理整除判断与奇偶筛选。
- 比较运算符的返回结果：`1`、`0` 与 `NULL`。
- `=`、`<=>`、`IS NULL`、`IS NOT NULL` 对 `NULL` 的处理方式。
- `BETWEEN AND`、`IN`、`NOT IN`、`LIKE`、`REGEXP` 在查询条件中的用法。

## 建议阅读顺序

1. 先看：[1 算术运算符](./1%20算术运算符.md)，掌握数值运算中最常见的行为。
2. 再看：[2 比较运算符](./2%20比较运算符.md)，理解 `WHERE` 条件中如何判断相等、范围、列表、空值与字符串模式。
3. 读完后，再回到第三章的 `SELECT` 与 `WHERE` 基础，把运算符放进实际查询语句中理解。
4. 后续补齐逻辑运算符等内容后，再按查询条件的复杂度逐步阅读。
