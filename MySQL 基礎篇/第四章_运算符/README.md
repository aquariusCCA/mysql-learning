# 第四章_运算符

这一章开始整理 MySQL 中的运算符。运算符是后续编写查询条件、计算表达式、筛选数据与组合逻辑判断的基础。

## 本章在整体学习地图中的位置

- 第三章主要解决的是 `SELECT`、`WHERE` 和表结构查看这些查询骨架问题。
- 这一章开始补上“条件和表达式到底怎么写”的细节，也就是把运算符真正放进查询语句里使用。
- 读完这一章后，建议接着进入 [第五章_排序与分页](../第五章_排序与分页/README.md)，把条件判断和结果展示串成完整查询流程。

## 建议阅读顺序

1. [1 算术运算符](./1%20算术运算符.md)：先掌握数值计算、除法、`DIV`、求模和 `NULL` 结果这些基础行为。
2. [2 比较运算符](./2%20比较运算符.md)：再理解 `=`、`<=>`、`IS NULL`、`BETWEEN AND`、`IN`、`LIKE`、`REGEXP` 等条件判断写法。
3. [3 逻辑运算符](./3%20逻辑运算符.md)：把 `NOT`、`AND`、`OR`、`XOR` 放进复合筛选条件里，理解真假判断与优先级。
4. [4 位运算符](./4%20位运算符.md)：最后理解 `&`、`|`、`^`、`~`、`<<`、`>>` 这组以二进制位为基础的运算方式。
5. [5 运算符的优先级](./5%20运算符的优先级.md)：最后统一回看不同运算符混用时的默认计算顺序，避免复杂表达式读错。
6. 读完后，再回到 [第三章_基本的SELECT语句](../第三章_基本的SELECT语句/README.md)，把运算符放回实际查询语句中理解。

## 本章小节

- [1 算术运算符](./1%20算术运算符.md)：理解加减乘除、`DIV`、求模与数值类型变化。
- [2 比较运算符](./2%20比较运算符.md)：理解相等、不等、空值判断、范围筛选、列表筛选与模式匹配。
- [3 逻辑运算符](./3%20逻辑运算符.md)：理解 `NOT`、`AND`、`OR`、`XOR` 的判断规则，以及 `NULL` 和优先级的影响。
- [4 位运算符](./4%20位运算符.md)：理解按位与、按位或、按位异或、按位取反，以及左右位移的规则。
- [5 运算符的优先级](./5%20运算符的优先级.md)：集中回查 MySQL 中多种运算符混用时的默认计算顺序。
