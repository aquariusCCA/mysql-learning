# 第四章_运算符

这一章开始整理 MySQL 中的运算符。运算符是后续编写查询条件、计算表达式、筛选数据与组合逻辑判断的基础。

目前已整理第 1 节，先从算术运算符开始，重点理解数值运算、类型变化、除以 `0` 的结果，以及取模在查询筛选中的用法。

## 本章主要内容

- 算术运算符的基本写法。
- 加法、减法、乘法、除法与求模的结果特点。
- MySQL 中 `+` 与字符串拼接的区别。
- 除法、`DIV`、除以 `0` 与 `NULL` 的关系。
- 使用 `MOD` 处理整除判断与奇偶筛选。

## 建议阅读顺序

1. 先看：[1 算术运算符](./1%20算术运算符.md)，掌握数值运算中最常见的行为。
2. 读完后，再回到第三章的 `SELECT` 与 `WHERE` 基础，把运算符放进实际查询语句中理解。
3. 后续补齐比较运算符、逻辑运算符等内容后，再按查询条件的复杂度逐步阅读。
