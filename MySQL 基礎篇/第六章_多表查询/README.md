# 第六章_多表查询

这一章开始把单表查询推进到“多个表如何一起查”。重点不只是学会 `JOIN` 语法，更重要的是建立多表连接的判断框架：什么时候会产生笛卡尔积、连接条件应该写在哪里、SQL92 和 SQL99 的写法有什么差别，以及不同 JOIN 图示分别该怎么落成 SQL。

## 本章在整体学习地图中的位置

- 第三章建立了 `SELECT`、`FROM`、`WHERE` 的基础查询骨架。
- 第四章补上了比较、范围、空值与逻辑判断所需的运算符。
- 第五章开始处理结果如何排序与分页。
- 这一章进一步进入“多个表之间怎么建立正确关系”，是后续子查询、聚合统计和复杂业务查询的重要前置基础。

- [前言](./前言.md)：先定义什么是多表查询，以及进行多表查询前表之间必须满足什么关系。
