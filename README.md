## 使用方式

- 想按課程順序閱讀：從上面的章節導航依序點進各節。
- 想快速查找主題：可直接使用 GitHub 搜尋，或利用每篇筆記開頭的 `关键字` 區塊定位內容。
- 想搭配圖片閱讀：各章圖片已集中在該章的 `images/` 目錄，Markdown 內皆使用相對路徑引用。
