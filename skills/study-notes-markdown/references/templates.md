# 模板

把這些當作模式參考，不要機械式硬套。

## 根目錄 README 模板

```md
# 倉庫標題

一句话说明仓库用途。

## 目前进度

- 已完成：

## 章节导航

### 课程名

#### 章节名

- [查看本章导航](./path/to/chapter/README.md)
- [1 小节标题](./path/to/section.md)

## 使用方式

- 顺序阅读怎么用
- 搜索关键字怎么用

## 参考资料
```

## 章節 README 模板

```md
# 第X章_章节名

这一章主要讲什么。

## 建议阅读顺序

1. 先看什么
2. 再看什么

## 本章小节

- [1 小节标题](./1%20小节标题.md)

## 本章适合回查的主题

- 某类问题看哪一节

## 返回总目录

- [回到 README](../../README.md)
```

## 單篇筆記開頭模板

```md
# 标题

> 所属章节：[章节名](./README.md)

## 本节导读

这一节主要说明……

如果你是第一次学习……；如果是复习……

## 关键字

- `术语`：简短说明
```

## 文末模板

```md
## 延伸阅读

- [相关笔记 A](./related-a.md)
- [相关笔记 B](./related-b.md)
```

## 連結檢查清單

加內部連結前，先檢查：

- 這是不是讀者自然會想去的下一站？
- 目標頁是否真的能解決後續問題？
- 這個連結是否比純文字更有價值？
- 這一段的連結是否已經太多？

如果前兩題答案是否，就不要加這個連結。
